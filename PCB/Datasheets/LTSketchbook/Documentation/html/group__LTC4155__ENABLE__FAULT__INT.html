Detailed Description

ENABLE_FAULT_INT Bit Field.

Enable OVP_ACTIVE, OTG_FAULT, BAD_CELL, and NTCSTAT = Hot_Fault interrupts.

  • Register: REG6
  • CommandCode: 0x06
  • Size: 1
  • Offset: 6
  • MSB: 6
  • MASK: 0x40
  • Access: R/W
  • Default: 0
