Detailed Description

REG5 Register.

7:1 0
NTCVAL NTC_WARNING
  • CommandCode: 0x05
  • Contains Bit Fields:
    • NTCVAL : NTC pin (thermistor) ADC conversion result.
    • NTC_WARNING : Thermistor temperature out of range for battery charging.
